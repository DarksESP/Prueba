

COMANDOS :
Nos muestra las ramas que tenemos creadas
git branch 

CREAMOS UNA RAMA NUEVA
git branch rama2


ELIMINAMOS rama2
LO USAMOS CUANDO YA HEMOS HECHO EL MERGE Y TENEMOS LOS DATOS GUARDADOS EN OTRA RAMA
git branch -d rama2

EN CAMBIO, USAMOS ESTE CUANDO QUEREMOS ELIMINAR DEFINITIVAMENTE LA RAMA SIN GUARDAR LOS DATOS
git branch -D rama3

Usamos este comando para cambiar de nombre las ramas 
b
git branch -m principal

Nos va a aparecer el nombre de rama, un nuevo identificatorio, y luego el utlimo commit
git branch -v 

PARA CAMBIAR DE RAMA
git checkout rama3

PARA CREAR Y CAMBIAR DE RAMA
git checkout -b estilos

PARA UNIFICAR LA INFORMACION
git merge estilos master

PRIMERO VA LA RAMA QUE TIENE LOS DATOS QUE QUEREMOS ENVIAR (estilos), y luego la rama que va a recibirlo (master).
